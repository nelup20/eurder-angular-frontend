So what's this demo project about?

Angular of course! More specifically: making a front-end for a Spring Boot application in Angular.

The order web app allows you to do the following for the various stories/sections:

  • Customers:

  • View all customers
  • Create a new customer
  • View the details of a specific customer
  • Edit and update a specific customer

  • Items:

  • View all items
  • Create a new item
  • View the details of a specific item
  • Edit and update a specific item

  • Orders:

  • Shop for items
  • View your basket of items
  • Place orders
